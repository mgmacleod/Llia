Adding Synths to Llia

Home Contents Synth Catalog Previous: Scenes Next: Adding Synth Tutorial

In broad outline adding a new synth to Llia requires the following steps:
  1. Define a SuperCollider synth.
  2. Create a Llia proxy for the new synth.
  3. Create logo and editor graphics

The addsynth script, located in the tools folder, may be used to automate much of this process. Quite a bit of manual editing will still be required but addsynth will create the basic framework and boilerplate code.

When calling addsynth you supply the synth name and type, and optionally a parameters list and target directory. If a synth with the same name already exists, addsynth will print a warning message and terminate. 

            ./addsynth --name <Name> --stype <type>
            
            Where <Name> is the synths name, which should be capitalized.
                  <type> is one of "synth", "efx" or "controller"

If executed from the tools directory addsynth infers the target directory as the current parent directory. You may pass an explicit target directory with the --dev option.

To automate the process further you may specify a parameter file.
            ./addsynth --name <Name> --stype <type> --params <filename>

A parameter list is best used for synths with relatively few parameters. The file structure has one parameter per line: 

            t param val
            t param val
               ......
            t param val
Where
  • t is the data type, either "f" for float or "i" for int.
  • param is the parameter name
  • val is the default value.
Do not include any of the following parameters:
  • outbus
  • inbus
  • xbus
  • doneAction
  • gate
  • freq
  • keynumber
  • detune
  • velocity
addsynth creates the following files (where <sname> is the synth name):
  • LLia/sc/SynthDefs/<sname>.scd
  • resources:
    • Llia/resources/<Sname>/editor.xcf
    • Llia/resources/<Sname>/editor.png
    • Llia/resources/<Sname>/tab.png
    • Llia/resources/<Sname>/logo.png
    • Llia/resources/<Sname>/logo_small.png
    • Llia/resources/<Sname>/logo_32.png
  • Llia synth proxy:
    • Llia/llia/synths/<sname>/__init__.py
    • Llia/llia/synths/<sname>/<sname>_proxy.py
    • Llia/llia/synths/<sname>/<sname>_data.py
    • Llia/llia/synths/<sname>/<sname>_pp.py
    • Llia/llia/synths/<sname>/<sname>_random.py
    • Llia/llia/synths/<sname>/tk/__init__.py
    • Llia/llia/synths/<sname>/tk/editor.py
The following files are modified:
  • Llia/sc/SynthDefs/install_synths.scd
  • Llia/llia/manifest.py
