Llia

Contents Synth Catalog Next: Install

Llia is an OSC client for SuperCollider providing a comprehensive MIDI implementation and comes pre bundled with several synthesizers and effects.

Features:
  • MIDI mapping to synth parameter.
  • Program banks.
  • Keysplits, layering and alternate scales.
  • Multiple key response modes.
  • Open collection of synths,effects and controllers.
  • Synth parameter editors.
  • Random program generators.
  • Synths may be patched graphically.
  • Python scripting and interactive REPL.

Obtaining:

Llia is available from Github at plewto/Llia.
