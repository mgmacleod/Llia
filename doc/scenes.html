A Scene is the current state of Llia.

Scene functions are accessible under the main window's File menu.
  • Save Scene.
    Saves the current state of Llia to a scene file. Scene files have the extension '.llia'
  • Load Scene.
    Reloads a previous scene file. Only a single scene may be resident at any one time.
    Loading a previous scene will reinitialize Llia, all unsaved data will be lost.
    Depending on their complexity, scenes may take several seconds to load.

Scenes are based on a previous feature called "Compose", and were developed in response to frustrations using Compose in an actual production environment. The main problem with Compose is that it relies on multiple data files in specific locations. Scene solves this problem by aggregating all data to a single file. Compose is still important because it provides the scripting background on which scenes are built.

Compose may still be accessed using the legacy "lliascript" option under the File menu.

WARNING: Scene files execute embedded Python code and have the same security concerns as lliascript. Only load trusted scene files.

The directory <install>/scenes/ contains two predefined scene files.
  • organ combines MASA organ and RDrum percussive synth with Scanner vibrato and reverb effects.
  • solalgo combines the Algo and Sol synths with the Fxstack effect and the Cascade pulse generator.
