This page contains an overview of known issues. It is not comprehensive and only contains bugs which are most likely to cause confusion. See the file Llia/BUGS for more details.

  • BUG 0014 Poly1 synths not being freed.
    As notes are played Poly1 creates server-side synth objects on the fly.
    As notes are released the corresponding synth should be freed once it's envelopes have ended.
    This is not always happening, at times a synth is never freed. Eventually these build
    up causing performance problems with the SC server.
    Currently it is not know exactly why these synths are never freed.

  • BUG 0017 Can not store program to slot 0.
    At times a program can not be stored into bank slot 0. It is unknown exactly when this bug crops up.
    Possible Workaround:
    1. Copy program to clipboard.
    2. Select program slot 0.
    3. Paste clipboard
    4. Store pasted program to slot 0.

  • BUG 0022 Restart does not correctly 'forget' previous synths.
    Workaround Exit and restart Llia client.
     
  • BUG 0026 Graph does not display audio path to output buses.
    This appears to be strictly a GUI issue and audio output works properly. It is not known exactly when the bug occurs but it first popped up after loading a scene file.
