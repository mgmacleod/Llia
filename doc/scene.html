A Scene provides a method for saving the current state of Llia for later recall. Scenes supersede the Compose feature.

The primary difference between Compose and Scene is that Scene saves all data to a single file. Compose is nothing more then a Python script. It makes an attempt to load bank data but each synth created by compose requires a separate bank data file. Worse if a bank file is moved Compose will be unable to locate it. Compose is still important as it provides the scripting portion for Scene objects.

Scenes are accessed by the "Save Scene" and "Load Scene" options under the main File menu. Depending on the complexity loading a Scene may take several seconds or a minute or two. Be aware that loading a Scene completely replaces the current state of Llia. All unsaved data is deleted.

WARNING: Scene files have the same security concerns as Compose. Only use scene files from trusted sources.

The directory Llia/scenes contains pre configured scene files.
