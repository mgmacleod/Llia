Llia Performances

In addition to synth parameters each Program contains a Performance which specifies the synth's response to specific MIDI events. The types of things a performance specifies are:

  • Transposition
  • Active key range
  • Pitch bend range
  • MIDI controller maps
  • MIDI pitch wheel maps
  • MID velocity maps
  • MIDI aftertouch maps
  • MIDI keynumber maps

Performance controls are spread across three tabs in an instrument editor.

Under the Performance tab there are 5 fields, two of these (MIDI channel and Key Table) are not actually part of the Performance. They are placed here for lack of a better location.

MIDI channel selects the MIDI input channel.

Key Table selects which key table to use. key tables define the relationship between MIDI key number and frequency and are used for alternate tunning. Currently only a few pre-defined tables are available. The table "EQ12" is the standard Western 12 note, equal tempered scale with A = 440Hz.

The Perf tab was relabeld as 'MIDI" after version 0.1.3.

Transpose sets note transposition in half steps. Transpose may be up to ±36 semi-tones (3 octaves).

Key Range sets the active key range in terms of lower and upper MIDI key numbers. Notes outside of this range are ignored. Key Range is useful for defining split keyboards.

Bend Range sets pitch bend amount in terms of cents The maximum range is 2400 cents (2 octaves).

Parameter Maps

Several MIDI values may be mapped to synth parameters. Typical examples are mapping modulation wheel to vibrato or key velocity to filter cutoff. With exception of MIDI continuous controllers, which are slightly more complex, all the mapping functions share a common interface. The Map1 tab is for MIDI controllers and Pitch Wheel. The Map2 tab contains Velocity, Aftertouch and Keynumber maps. All maps have the following values:

  • Source signal
  • Destination parameter
  • Curve
  • Curve modifier
  • Range
  • Limits

Source signal really only applies to MIDI controllers. The source is implicit (Velocity, Keynumber etc) for the other control sources. For MIDI controllers either the controller number is specified or a symbolic controller name as defined in the config file.

Destination parameter is the synth parameter the source is mapped to. Available parameters are provided in a drop down list. The parameter names can at times be rather cryptic. These names are usually not exact matches for the labels on the synth editors. You can however always tell what the parameter (and current value) associated with an editor control is by moving the mouse over it. The parameter and value will appear in the status line at the bottom of the editor window.

Curve selects the mapping function between the source and parameter. Available curves are:

  • Linear
  • Exponential
  • Logistic or "S"
  • Step

Curve Modifier is a coefficient which modifies the curve in some way. The exact effect is dependent on the curve type. For linear curves the modifier has no effect. For exponential and logistic curves the modifier alters the curve steepness. The modifier sign changes the exponential curve concavity. A positive value produce a concave curve while negative values are convex. or maybe its the other way around?. For the step curve the modifier sets the number of steps. Negative step counts alters the distribution to include the end points.

Range sets the parameter range. Many, but by no means all, parameters are "normalized" to take values between 0 and 1. If the high and low values are inverted, such that low > high, the mapped curve will also be inverted.

Limits sets an absolute limit on the parameter value. If the range is greater then the limit then some portion of curve will be clipped.

You add a new parameter map by clicking the [+] button in the appropriate section. Similarly a parameter map is removed by clicking [-]. When removing a mapping the special value "ALL" removes all maps from the give source.

In the case of keynumber the mapping function is aware of the current key range and takes that into account when calculating the parameter value. The low key in the range has a value of 0.0, the high key has value 1.0.

Pitchwheel maps are in addition to the normal pitch bend functions. To defeat normal pitch bending set the bend range to 0. Pitchwheel is the only source which may have a negative value.
